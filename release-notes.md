## Docker

```bash
# CPU only
docker pull ghcr.io/nomercy-entertainment/nomercymediaserver:latest
docker pull ghcr.io/nomercy-entertainment/nomercymediaserver:0.1.249

# NVIDIA GPU
docker pull ghcr.io/nomercy-entertainment/nomercymediaserver:nvidia

# Intel Quick Sync
docker pull ghcr.io/nomercy-entertainment/nomercymediaserver:intel

# AMD GPU
docker pull ghcr.io/nomercy-entertainment/nomercymediaserver:amd
```

Or use Docker Compose — see `docker-compose.yml` (and GPU variants) in the repo.
